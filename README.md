<h1>CALCULADORA HEXADECIMAL</h1>
Calculadora Hexadecimal de números inteiros.

<h2>Como inicializar:</h2>
<ol>
  <li>Instale Flutter na sua máquina</li>
  <li>Clone esse repositório</li>
  <li>cd na pasta</li>
  <li>flutter run</li>
  <li>Escolha o emulador</li>
</ol>
